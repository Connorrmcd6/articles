# Recharts MDX Sample

Below are a few basic charts rendered with custom wrapper components that use Recharts.

## Line Chart - Revenue vs Expenses

<CustomLineChart
  title="Monthly Financial Performance"
  description="Revenue and expenses comparison for Q1 2024"
  data={[
    { month: "Jan", revenue: 12000, expenses: 8000 },
    { month: "Feb", revenue: 15000, expenses: 9000 },
    { month: "Mar", revenue: 18000, expenses: 11000 },
    { month: "Apr", revenue: 16000, expenses: 10000 },
  ]}
  xAxis="month"
  yAxis={["revenue", "expenses"]}
/>

You can see the revenue trend is growing steadily while expenses remain relatively stable.

## Bar Chart - Regional Sales

<CustomBarChart
  title="Sales by Region"
  description="Total sales across different regions"
  data={[
    { region: "North", sales: 25000 },
    { region: "South", sales: 18000 },
    { region: "East", sales: 30000 },
    { region: "West", sales: 22000 },
  ]}
  xAxis="region"
  yAxis="sales"
/>

The East region shows the strongest performance with 30,000 in sales.

## Area Chart - Growth Over Time

<CustomAreaChart
  title="Cumulative Growth"
  description="Revenue growth visualization"
  data={[
    { quarter: "Q1", revenue: 50000, profit: 12000 },
    { quarter: "Q2", revenue: 75000, profit: 20000 },
    { quarter: "Q3", revenue: 95000, profit: 28000 },
    { quarter: "Q4", revenue: 120000, profit: 35000 },
  ]}
  xAxis="quarter"
  yAxis={["revenue", "profit"]}
/>

Area charts are great for showing cumulative trends and growth patterns.

## Pie Chart - Product Distribution

<CustomPieChart
  title="Product Market Share"
  description="Distribution of sales by product category"
  data={[
    { name: "Product A", value: 400 },
    { name: "Product B", value: 300 },
    { name: "Product C", value: 200 },
    { name: "Product D", value: 100 },
  ]}
  nameKey="name"
  dataKey="value"
/>

Product A dominates the market share at 40%.

## Multiple Y-Axes

You can plot multiple metrics on the same chart by passing an array to the `yAxis` prop:

<CustomBarChart
  title="Multi-Metric Comparison"
  data={[
    { category: "A", metric1: 100, metric2: 200, metric3: 150 },
    { category: "B", metric1: 150, metric2: 180, metric3: 220 },
    { category: "C", metric1: 200, metric2: 220, metric3: 180 },
  ]}
  xAxis="category"
  yAxis={["metric1", "metric2", "metric3"]}
/>

That's it! Simple, clean, and powerful data visualization in MDX.
